# Customer Subscription EDA & Revenue Insights

## Project Overview

This project analyzes customer subscription data for a fictional telecommunications company to identify churn patterns, revenue risk, customer segments, and business opportunities for improving retention.

The goal is to simulate a realistic data analyst workflow using Python, SQL, Excel, and dashboard planning.

## Business Questions

1. Which customer segments have the highest churn rates?
2. Which contract types and payment methods are associated with higher churn?
3. How much monthly revenue is at risk from churned customers?
4. Which customer groups generate the most recurring revenue?
5. What business actions could reduce churn and improve retention?

## Tools Used

- Python
- Pandas
- NumPy
- SQL
- Excel
- Tableau or Power BI
- GitHub

## Project Workflow

1. Data cleaning
2. Exploratory data analysis
3. SQL revenue and churn analysis
4. Excel validation and pivot tables
5. Dashboard planning
6. Business insights and recommendations

## Dataset

This project uses a customer subscription churn dataset containing customer demographics, account information, services, billing details, and churn status.
